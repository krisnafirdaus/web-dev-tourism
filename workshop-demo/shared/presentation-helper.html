Struktur tabel utama:

-- Tabel Hotels CREATE TABLE hotels ( hotel_id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(255) NOT NULL, address TEXT NOT NULL, star_rating DECIMAL(2,1), amenities JSON, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ); -- Tabel Rooms CREATE TABLE rooms ( room_id INT PRIMARY KEY AUTO_INCREMENT, hotel_id INT NOT NULL, room_type VARCHAR(100) NOT NULL, price_per_night DECIMAL(12,2) NOT NULL, capacity INT DEFAULT 2, FOREIGN KEY (hotel_id) REFERENCES hotels(hotel_id) ); -- Tabel Bookings CREATE TABLE bookings ( booking_id INT PRIMARY KEY AUTO_INCREMENT, guest_id INT NOT NULL, room_id INT NOT NULL, check_in DATE NOT NULL, check_out DATE NOT NULL, status ENUM('pending', 'confirmed', 'cancelled') DEFAULT 'pending' );

üìã Quick Reference Cheat Sheet

HTML
HyperText Markup Language - struktur dasar halaman web
CSS
Cascading Style Sheets - styling dan layout halaman web
JavaScript
Bahasa pemrograman untuk interaktivitas di browser
DOM
Document Object Model - representasi struktur HTML yang bisa dimanipulasi dengan JS
API
Application Programming Interface - cara komunikasi antara frontend dan backend
REST
Representational State Transfer - arsitektur API berbasis HTTP methods
JSON
JavaScript Object Notation - format data untuk pertukaran informasi
SQL
Structured Query Language - bahasa untuk mengelola database
ORM
Object-Relational Mapping - teknik mapping object ke database table
JWT
JSON Web Token - token untuk autentikasi user
OAuth
Open Authorization - protokol untuk akses delegated
Webhook
HTTP callback yang dipicu oleh event tertentu
Race Condition
Kondisi dimana hasil bergantung pada timing/timing event
ACID
Atomicity, Consistency, Isolation, Durability - properties database transaction
CORS
Cross-Origin Resource Sharing - mekanisme untuk request antar domain
HTTPS
HTTP Secure - protokol HTTP dengan enkripsi SSL/TLS
